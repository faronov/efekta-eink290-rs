Encoding Format All sprites are 1bpp row-packed MSB-first. Header: 0x01 0x01 W_lo W_hi H_lo H_hi.
Each row is ceil(width/8) bytes. Bit=1 → foreground pixel, Bit=0 → background.
Total payload: ceil(width/8) × height bytes per sprite.
